# Fitness Sports Center Website

A modern, responsive landing page for a fitness center built with React and Tailwind CSS.

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS 3
- PostCSS + Autoprefixer

## Setup Instructions

### 1) Clone the repository

```bash
git clone <your-repo-url>
cd website
```

### 2) Install dependencies

```bash
npm install
```

### 3) Start development server

```bash
npm run dev
```

### 4) Build for production

```bash
npm run build
```

### 5) Preview production build

```bash
npm run preview
```
